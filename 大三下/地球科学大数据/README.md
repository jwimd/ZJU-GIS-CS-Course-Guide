这门课我多少沾点私人恩怨，先不提这讲座课的含金量吧（懂的都懂），我跟这门课就是八字不合

- 选课时因为和编译原理考试冲突，导致我只能明年修编译原理，而考虑到方向和夏令营的问题我只能自学了编译原理

导致我对这门课印象不好，~~狠狠低分~~
